# ICMAR Unsupervised Machine Learning Analysis

## Overview
Dimensionality reduction and clustering analysis on ICMR health dataset using PCA, LDA, and t-SNE techniques.

## Objectives
- Compare dimensionality reduction techniques
- Identify patterns in high-dimensional medical data
- Visualize clusters and data structure

## Techniques
- **PCA**: Variance-based dimensionality reduction
- **LDA**: Supervised dimensionality reduction
- **t-SNE**: Non-linear dimensionality reduction

## Technologies
- Python 3.x
- Scikit-learn for ML algorithms
- Pandas, NumPy for data handling
- Matplotlib, Seaborn for visualization

## Methodology
1. Data preprocessing and standardization
2. PCA analysis with scree plot
3. LDA for supervised reduction
4. t-SNE for non-linear visualization
5. Cluster comparison and interpretation

## Key Findings
- Variance explained by principal components
- Class separation in reduced dimensions
- Cluster patterns in medical data

## How to Run
```bash
jupyter notebook ICMAR_dataset_Unsupervised_ML_Analysis.ipynb
```

# ICMAR-dataset-Unsupervised-ML-Analysis
Unsupervised ML analysis on ICMR health dataset using PCA, LDA &amp; t-SNE. Includes data preprocessing, standardization, dimensionality reduction techniques, and scree plot visualization. Compares variance-based (PCA), supervised (LDA), and non-linear (t-SNE) methods for pattern discovery and cluster visualization in medical data.
